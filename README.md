# Movie SPA

## Описание

SPA для просмотра фильмов с фильтрацией, избранным и сравнением

## Функционал

- Список фильмов с бесконечным скроллом
- Фильтры (жанры, рейтинг, год)
- Детальная страница фильма
- Избранное
- Сравнение фильмов

## Стек

- React
- TypeScript
- React Router
- Axios
- Effector
- PoisKino API
- CSS Modules

## Установка

```bash
git clone https://github.com/VikaGovorina/kinopoisk-spa
npm install
```

## Переменные окружения

Создайте файл `.env` на основе `.env.default` и укажите API ключ:

VITE_API_KEY=your_api_key

## Запуск

```bash
npm run dev
```
